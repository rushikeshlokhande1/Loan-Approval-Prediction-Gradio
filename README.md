# 💳 Loan Approval Prediction App

This project is a Machine Learning-based Loan Approval Prediction System 
built using **Gaussian Naive Bayes** and deployed with an interactive 
**Gradio Web Interface**.

---

## 🚀 Project Overview

The model predicts whether a loan application will be:

✅ Approved  
❌ Rejected  

based on applicant financial and demographic details.

---

## 🧠 Machine Learning Model

- Algorithm: **Gaussian Naive Bayes**
- Preprocessing:
  - One-Hot Encoding
  - Feature Scaling (StandardScaler)

---

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Gradio

---

## 📊 Features Used

- Applicant Income
- Coapplicant Income
- Age
- Dependents
- Credit Score
- Existing Loans
- DTI Ratio
- Savings
- Collateral Value
- Loan Amount
- Loan Term
- Education Level
- Employment Status
- Marital Status
